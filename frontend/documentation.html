Getting Started

Introduction

Gene Expression Explorer is a web-based tool for analyzing and visualizing gene expression data from NCBI's GEO repository.

Quick Setup

  1. Enter a GEO dataset ID (e.g., GSE1009)
  2. Select analysis parameters
  3. View and download results

API Documentation

Endpoints

GET /api/search_dataset

Retrieve dataset information from GEO.


GET /api/search_dataset?gse_id=GSE1009

POST /api/analyze

Perform analysis on dataset.


POST /api/analyze
{
    "dataset_id": "GSE1009",
    "analysis_type": "differential"
}

Frequently Asked Questions

What types of data can I analyze?

You can analyze any gene expression dataset available in the GEO repository, including microarray and RNA-seq data.

How do I cite this tool?

Please cite our paper [Reference] when using Gene Expression Explorer in your research.

Can I analyze my own data?

Yes, you can upload your own expression data in various formats (CSV, TSV, GCT).
